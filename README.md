# Portfolio scaffold

This repository contains a minimal portfolio website scaffold (HTML, CSS, JS).

Files
- `index.html` — semantic outline with sections: header, hero, about, projects, skills, contact, footer.
- `style.css` — baseline styles and responsive layout.
- `script.js` — mobile nav toggle, smooth scrolling, and a placeholder contact form handler.

How to use
1. Open `index.html` in your browser (double-click or use a local server).
2. Replace placeholder text (Your Name, bio, project descriptions, email).
3. Wire contact form to your backend or a form service.

Next improvements
